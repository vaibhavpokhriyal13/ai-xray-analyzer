# AI Chest X-Ray Analyzer

An AI-powered chest X-ray analysis system built using TensorFlow and Streamlit that detects possible pneumonia from chest X-ray images.

## Features

- Chest X-ray upload
- Pneumonia detection
- Confidence score
- AI-generated medical report
- Interactive Streamlit dashboard
- Deep learning using MobileNetV2

## Tech Stack

- Python
- TensorFlow
- Streamlit
- MobileNetV2
- NumPy
- Pillow

## Project Workflow

1. Upload chest X-ray image
2. Image preprocessing
3. Deep learning prediction
4. Confidence score generation
5. AI medical report display

## Model

Transfer learning was implemented using MobileNetV2 pretrained on ImageNet.

## Disclaimer

This project is for educational purposes only and not intended for real medical diagnosis.
